# 研究方法

## 模型设定与假设

本研究的模拟框架建立在一系列旨在捕捉教育干预核心机制的数学设定与理论假设之上。这些设定共同构成了一个可计算的虚拟环境，用以检验“诊断-匹配”干预策略的优越性。

### 1. 学生成就生成模型

我们假设学生的学业成就（Achievement, A）由其潜在的知识水平（Knowledge, K）和动机水平（Motivation, M）两个核心因素共同决定。这一关系通过一个加权线性模型进行形式化表达：

`A = α * K_norm + β * M_norm`

在此模型中，`K_norm` 与 `M_norm` 是经过最小-最大标准化处理后、值域在 [0, 1] 区间的知识与动机得分。`α` 和 `β` 分别是知识与动机的权重系数，代表二者对最终学业成就的相对贡献度，且满足 `α + β = 1`。此设定构成了学生异质性的基础，使得依据 K 和 M 的不同组合来诊断学生短板成为可能。

### 2. 干预效应的交互作用机制

本模型的核心假设在于，干预措施的效果并非恒定，而是与学生的内在特质存在交互作用。具体而言，知识技能型干预的效果受到学生动机水平的调节。

*   **技能干预 (Skill Intervention)**：旨在提升知识水平，其效果由以下函数决定：
    `ΔK_skill = δ_K * (1 + γ * M_norm)`
    其中，`ΔK_skill` 为知识增量，`δ_K` 为基础知识增益，代表干预本身的基线效果。关键参数 `γ`（动机-知识交互系数）若大于零，则意味着学生当前的动机水平 `M_norm` 会正向放大干预效果。这一交互效应是“匹配增益”得以在模型中涌现的根本机制。

*   **动机干预 (Motivation Intervention)**：旨在提升动机水平，其效果被设定为一个固定增量：
    `ΔM_motivation = δ_M`
    其中，`δ_M` 是一个常数，代表动机干预所能带来的固定提升。

*   **通用干预 (General Intervention)**：模拟“一刀切”的普适性方案，其效果被设定为在两个维度上均产生较弱的固定增益：
    `ΔK_general = δ_K_gen`
    `ΔM_general = δ_M_gen`
    为体现其非针对性，参数设定上满足 `δ_K_gen < δ_K` 且 `δ_M_gen < δ_M`。

## 模拟数据生成

### 1. 初始总体的生成与设定

我们生成一个包含 N=10,000 名学生的初始总体。每名学生的潜在特质（K, M）通过从一个二维正态分布中抽样获得：

`[K, M] ~ MVN(μ = [0, 0], Σ)`

其中，均值向量 `μ` 为 `[0, 0]`，协方差矩阵 `Σ` 被设定为 `[[1, 0.2], [0.2, 1]]`。一个较小的正相关系数 `ρ = 0.2` 被引入，以反映知识与动机在现实中存在的微弱正向关联。随后，通过最小-最大缩放将 K 和 M 标准化为 `K_norm` 和 `M_norm`。基于 `α = 0.6` 和 `β = 0.4` 的权重，计算每位学生的初始学业成就 `A_pre`。

### 2. 低成就学生的筛选与诊断

首先，我们将初始学业成就 `A_pre` 排序，并将分数处于后20个百分位（bottom quintile）的学生界定为“低成就学生”，构成后续干预实验的目标样本。

其次，为实施诊断，我们仅在低成就学生群体内部计算其 `K_norm` 和 `M_norm` 的均值（`μ_K_low`, `μ_M_low`）与标准差（`σ_K_low`, `σ_M_low`）。基于此，学生被划分为三种类型：
*   **知识短板型**：`K_norm < μ_K_low - 0.5 * σ_K_low` 且 `M_norm ≥ μ_M_low - 0.5 * σ_M_low`。
*   **动机短板型**：`M_norm < μ_M_low - 0.5 * σ_M_low` 且 `K_norm ≥ μ_K_low - 0.5 * σ_K_low`。
*   **混合短板型**：`K_norm < μ_K_low - 0.5 * σ_K_low` 且 `M_norm < μ_M_low - 0.5 * σ_M_low`。

## 实验设计与干预流程

本研究采用多臂随机对照试验设计。所有被识别为“低成就”的学生将被随机且均等地分配至以下五个实验条件之一。

1.  **控制组 (Control)**：不接受任何干预。其干预后状态 `K_post` 和 `M_post` 与干预前相同。
2.  **通用干预组 (General)**：所有学生均接受标准化的通用干预，其知识和动机水平分别增加 `δ_K_gen` 和 `δ_M_gen`。
3.  **纯技能干预组 (Skill-Only)**：所有学生均接受技能干预。其知识水平根据 `K_post = K_pre + δ_K * (1 + γ * M_norm_pre)` 进行更新，动机水平不变。
4.  **纯动机干预组 (Motivation-Only)**：所有学生均接受动机干预。其动机水平增加 `δ_M`，知识水平不变。
5.  **诊断-匹配干预组 (Diagnosis-Matching)**：根据学生的诊断分类施加针对性干预：
    *   **知识短板型**：仅接受技能干预。
    *   **动机短板型**：仅接受动机干预。
    *   **混合短板型**：同时接受技能干预与动机干预。

干预结束后，所有学生的最终学业成就 `A_post` 将依据更新后的 `K_post` 和 `M_post` 重新计算。

## 数据分析与评价指标

### 1. 因变量与评价指标

*   **成就增益 (Gain Score)**：作为本研究的核心因变量，定义为 `Gain_Score = A_post - A_pre`。
*   **匹配增益 (Matching Gain)**：作为量化策略优越性的核心指标，定义为“诊断-匹配干预组”与“通用干预组”的平均成就增益之差：`Matching_Gain = Mean(Gain_Score_Matching) - Mean(Gain_Score_General)`。
*   **效应量 (Effect Size)**：采用科恩的d（Cohen's d）来量化关键组间比较的标准化差异，如“诊断-匹配组”与“通用干预组”之间的差异。

### 2. 统计检验

首先，将以“干预条件”为自变量，“成就增益”为因变量，进行一次单因素方差分析（One-Way ANOVA），以检验各组间是否存在显著差异。若ANOVA结果显著，则继续执行一系列预先规划的独立样本t检验，用于成对比较，包括但不限于：
*   “诊断-匹配组” vs. “通用干预组”
*   “诊断-匹配组” vs. “纯技能干预组”
*   “诊断-匹配组” vs. “纯动机干预组”
*   各干预组 vs. “控制组”

### 3. 数据可视化

研究结果将通过两类图表进行可视化呈现：
1.  **学生异质性散点图**：以 `K_norm` 为x轴，`M_norm` 为y轴，绘制初始总体的分布，并用不同颜色突出显示低成就学生群体，直观展示其内部的异质性。
2.  **成就增益箱线图**：以“干预条件”为x轴，“成就增益”为y轴，绘制箱线图。该图将直观对比五个实验条件下成就增益的分布（中位数、四分位距、离散程度），并标注出具有统计显著性的组间差异。
